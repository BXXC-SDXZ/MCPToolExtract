## Compatibilidad con ediciones de GitLab

GitLab MCP Server funciona tanto con Community Edition (CE) como con Enterprise Edition (EE). El catálogo Enterprise/Premium desbloquea 16 meta-herramientas adicionales exclusivas de Enterprise: usa `GITLAB_ENTERPRISE=true` en modo stdio, o `--enterprise`/autodetección de edición en modo HTTP.

| Funcionalidad                            | Community (CE) | Enterprise (EE) |
| ---------------------------------------- | :------------: | :-------------: |
| Proyectos, Issues, MRs, Pipelines, CI/CD |       ✅       |       ✅        |
| Wikis, Etiquetas, Hitos, Releases        |       ✅       |       ✅        |
| Usuarios, Grupos, Miembros, Búsqueda     |       ✅       |       ✅        |
| Despliegues, Entornos, Paquetes          |       ✅       |       ✅        |
| 33 meta-herramientas base                |       ✅       |       ✅        |
| 46 recursos, 37 prompts                  |       ✅       |       ✅        |
| Merge Trains                             |       ❌       |       ✅        |
| Métricas DORA                            |       ❌       |       ✅        |
| Gestión de vulnerabilidades              |       ❌       |       ✅        |
| Eventos de auditoría                     |       ❌       |       ✅        |
| Políticas de cumplimiento                |       ❌       |       ✅        |
| +16 meta-herramientas enterprise         |       ✖        |       ✅        |

Para habilitar funcionalidades enterprise en modo stdio, configura `GITLAB_ENTERPRISE=true`. En modo HTTP, configura `--enterprise` para forzar el catálogo Enterprise/Premium, u omítelo para que el servidor autodetecte CE/EE por entrada token+URL cuando GitLab informe la edición.

## Soporte de plataformas

Hay binarios precompilados disponibles para todas las plataformas principales:

| SO      | Arquitectura          | Binario                               |
| ------- | --------------------- | ------------------------------------- |
| Linux   | amd64                 | `gitlab-mcp-server-linux-amd64`       |
| Linux   | arm64                 | `gitlab-mcp-server-linux-arm64`       |
| macOS   | amd64 (Intel)         | `gitlab-mcp-server-darwin-amd64`      |
| macOS   | arm64 (Apple Silicon) | `gitlab-mcp-server-darwin-arm64`      |
| Windows | amd64                 | `gitlab-mcp-server-windows-amd64.exe` |
| Windows | arm64                 | `gitlab-mcp-server-windows-arm64.exe` |

## Compatibilidad con clientes MCP

Cualquier cliente que soporte el transporte stdio del Model Context Protocol puede usar este servidor. Clientes probados:

| Cliente                           | Transporte | Estado |
| --------------------------------- | ---------- | :----: |
| VS Code + GitHub Copilot          | stdio      |   ✅   |
| Claude Desktop                    | stdio      |   ✅   |
| Cursor                            | stdio      |   ✅   |
| Claude Code (CLI)                 | stdio      |   ✅   |
| Windsurf                          | stdio      |   ✅   |
| JetBrains IDEs                    | stdio      |   ✅   |
| Zed                               | stdio      |   ✅   |
| Kiro                              | stdio      |   ✅   |
| Cualquier cliente Streamable HTTP | HTTP       |   ✅   |

## Funcionalidades del protocolo MCP

| Capacidad   |                                  Soportada                                  |
| ----------- | :-------------------------------------------------------------------------: |
| Tools       | ✅ (hasta 1033 individuales / 33 base, 49 autoalojadas, 50 GitLab.com meta) |
| Resources   |                                   ✅ (46)                                   |
| Prompts     |                                   ✅ (37)                                   |
| Completions |                         ✅ (17 tipos de argumento)                          |
| Sampling    |                                     ✅                                      |
| Elicitation |                                     ✅                                      |
| Roots       |                                     ✅                                      |
| Logging     |                                     ✅                                      |
| Progress    |                                     ✅                                      |

La cifra de 1033 herramientas individuales es el conjunto expandido de instancias de herramienta distintas. Los recuentos de 33 base, 49 autoalojadas y 50 en GitLab.com son tamaños del catálogo de meta-herramientas cuyas acciones se expanden a esa superficie individual mayor.
