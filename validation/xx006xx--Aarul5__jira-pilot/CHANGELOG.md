# Changelog

All notable changes to this project will be documented in this file.

## [2.2.0] - 2026-02-13

### Added
- **MCP Prompts**: Added `jira-assist` and `jira-summarize-issue` prompts to guide LLMs.
- **MCP Resources**: Added `jira://myself` and `jira://projects` resources for direct context access.
- **MCP Capabilities**: Explicitly advertised `prompts` and `resources` capabilities.
- **Verification Script**: Added `scripts/verify-mcp.ts` for automated server verification.

### Changed
- **MCP Server**: Updated to use `2.2.0` version in server initialization.
- **README**: Updated with Prompts and Resources documentation.
