# Structural Analysis

Domain expertise for interpreting ReNoUn 17-channel structural analysis results.

## When This Applies

When a user discusses conversation quality, structural health, stuck loops, pattern detection, dialogue dynamics, prompt drift, interaction patterns, or structural coherence in any turn-based data.

## The 17 Channels

ReNoUn measures three dimensions, each decomposed into subchannels (all values 0.0-1.0):

**Recurrence (Re₁-Re₅)** — Return of the same formal pattern:
- Re₁ Lexical: vocabulary recycling
- Re₂ Syntactic: sentence structure repetition
- Re₃ Rhythmic: turn length/pacing consistency
- Re₄ Turn-Taking: predictability of who speaks when
- Re₅ Self-Interruption: repeated self-correction patterns

**Novelty (No₁-No₆)** — Measurable interruption of established patterns:
- No₁ Lexical: new vocabulary entering
- No₂ Syntactic: new sentence structures
- No₃ Rhythmic: sudden pacing changes
- No₄ Turn-Taking: unexpected interaction shifts
- No₅ Self-Interruption: new correction patterns
- No₆ Global Vocabulary Rarity: statistically rare words

**Unity (Un₁-Un₆)** — Degree of coherence:
- Un₁ Lexical Cohesion: vocabulary linking across turns
- Un₂ Syntactic Cohesion: consistent grammar
- Un₃ Rhythmic Cohesion: pacing consistency
- Un₄ Interactional Cohesion: stable interaction pattern
- Un₅ Anaphoric Cohesion: building on prior turns
- Un₆ Global Structural Symmetry: first-half/second-half mirroring

## DHS Interpretation

| Range | Assessment | Action |
|-------|-----------|--------|
| 0.00-0.35 | Distressed | Flag immediately |
| 0.35-0.55 | Below baseline | Note concern, check which dimension is low |
| 0.55-0.75 | Healthy | Report normally |
| 0.75-1.0 | Excellent | Highlight what's working |

## Key Principle

ReNoUn measures STRUCTURE, not CONTENT. Every pattern has multiple possible interpretations. Report structural findings clearly, present possible interpretations, and let the user apply domain-specific meaning. Never assert a single interpretation from structural data alone.
